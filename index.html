Cara Menggunakan Aplikasi

üéØ Fitur Baru: Background Monitoring!

Aplikasi sekarang berjalan di system tray. Tutup window tidak akan menghentikan monitoring - aplikasi tetap aktif di background!

1
Pilih Folder yang Akan Dipantau

Klik tombol "Pilih Folder" dan pilih folder tempat foto-foto akan disimpan (misalnya folder DCIM dari kamera)

2
Atur Konfigurasi API

Masukkan API URL server Anda. Jika diperlukan, tambahkan Bearer Token dan Session Code untuk identifikasi

3
Mulai Monitoring

Klik tombol "Mulai" untuk mengaktifkan monitoring otomatis. Setiap foto baru yang ditambahkan ke folder akan otomatis terupload

4
Monitor Proses Upload

Pantau log aktivitas. Foto akan diupload otomatis satu per satu ke server

Tips Penggunaan

  • Background Monitoring: Close window untuk minimize ke tray - monitoring tetap aktif!
  • Quit Aplikasi: Klik kanan icon di tray ‚Üí pilih "Quit" untuk benar-benar menutup
  • Auto-Resume: Queue tersimpan otomatis - tidak hilang saat restart app
  • Periodic Scan: Folder di-scan otomatis setiap 30 detik untuk memastikan tidak ada file terlewat
  • Pastikan Laravel backend sudah running sebelum memulai
  • Session code akan auto-generate jika dikosongkan
  • Gunakan "Reset History" untuk membersihkan data statistik
  • Format foto yang didukung: JPG, JPEG, PNG, GIF, WEBP
