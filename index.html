Pengaturan Laravel API

Endpoint Laravel untuk upload foto
Generate token di Laravel dengan tinker
Kosongkan untuk auto-generate berdasarkan timestamp

Laravel API Setup

1
Pastikan Laravel Running

php artisan serve

2
Generate API Token

php artisan tinker
$token = App\Models\ApiToken::generate('Token');
echo $token->token;

3
Test API

Upload akan otomatis ke /api/photos/upload

Tips

  • API URL default: http://localhost:8000/api/photos/upload
  • Session code auto-generate jika kosong
  • Token optional untuk testing, wajib untuk production
  • Check Laravel logs jika error: storage/logs/laravel.log
